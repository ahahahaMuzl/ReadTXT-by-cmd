### 启动说明
```
:: 默认打开"book.txt"
X:\ReadTXT.bat 

:: 打开"X:\盗墓笔记.txt"
X:\ReadTXT.bat  X:\盗墓笔记.txt
```

### 操作说明
```
-------------------------------------
c       清除书签
f       查找（按f后直接键入查找内容）
h       帮助文档
空格    清除当前屏幕
翻页控制 ↑↓←→ wsad Enter
-------------------------------------
```

### 简单参数
```
文件：FunctionMain.ps1
内容：
$showLineCnt = 5        # 每次显示行数
$showLineSize = 50      # 每次显示字数
```
